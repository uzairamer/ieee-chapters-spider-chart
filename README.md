# IEEE Chapters Spider Chart

## How to make changes?
  1) Download [FreeMind](http://freemind.sourceforge.net/wiki/index.php/Main_Page)
  2) Clone the repo
  3) Make changes in the IEEE Chapters.mm file and produce a pdf version as well.
  4) Make a PR
